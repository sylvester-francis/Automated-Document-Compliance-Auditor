About the Document Compliance Auditor

Project Overview

The Automated Document Compliance Auditor is a tool designed to help organizations ensure their documents comply with various regulations such as GDPR and HIPAA. It uses a combination of rule-based pattern matching and AI-powered suggestions to identify missing clauses and suggest remediation.

Key Features
  • Document Processing: Extract text from PDF, DOCX, and TXT files
  • Compliance Checking: Verify documents against GDPR, HIPAA, and other compliance standards
  • AI Suggestions: Generate remediation suggestions using advanced language models
  • Interactive UI: Highlight issues directly in the document with real-time feedback
Technologies Used
  • Backend: Python with Flask
  • Database: MongoDB for document storage
  • Text Extraction: PyPDF2, python-docx
  • AI Integration: OpenAI API
  • Frontend: HTML, CSS, JavaScript with HTMX for interactivity
